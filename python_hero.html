Privacy Policy – Python Hero

Effective Date: July 7, 2025

Thank you for using Python Hero, an educational mobile application designed to help users learn and master Python programming.

This Privacy Policy explains how we collect, use, and protect your information when you use the app.

1. Information We Collect

Python Hero is designed to prioritize your privacy. We do not collect any personal information from users.

However, we may collect limited non-personal data, such as:

  • Device type and OS version (for debugging and analytics)
  • Anonymous usage statistics (e.g., which lessons are most accessed)

This data is collected solely to improve the app and user experience.

2. How We Use Information

Any data we collect is used strictly for:

  • Improving app performance and usability
  • Fixing bugs or issues
  • Understanding usage patterns to improve learning content

We do not use your data for advertising, tracking, or sharing with third parties.

3. Third-Party Services

Python Hero does not integrate with third-party services such as AdMob, Google Analytics, or social media SDKs. Therefore, no personal or behavioral data is shared externally.

If any third-party services are introduced in future updates, this policy will be updated accordingly.

4. Data Security

We take reasonable steps to secure any data collected from unauthorized access or disclosure. Since Python Hero does not store any personal data or credentials, the risk is minimal.

5. Children's Privacy

Python Hero is suitable for users of all ages and does not knowingly collect data from children under 13. If you are a parent or guardian and believe your child has provided any data, please contact us for removal.

6. Changes to This Policy

We may update this Privacy Policy as needed. Users will be notified of any significant changes via the app or an in-app message.
